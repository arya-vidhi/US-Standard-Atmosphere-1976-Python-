# US Standard Atmosphere 1976 (Python)

A Python implementation of the U.S. Standard Atmosphere 1976 model for computing atmospheric properties over a wide range of altitudes.

This project implements the piecewise atmospheric model using the standard lapse rates and atmospheric layer definitions specified in the U.S. Standard Atmosphere 1976.

## Features

- Temperature calculation
- Pressure calculation
- Density calculation
- Geopotential altitude conversion
- Atmospheric layer handling
- Atmospheric property visualization using Matplotlib

## Files

```
US Standard Atmosphere 1976.py
```

Main program that computes atmospheric properties and generates plots.

```
US 1976 functions.py
```

Contains reusable functions for atmospheric property calculations.

## Libraries Used

- NumPy
- Matplotlib
- Pandas
- SciPy

The program computes atmospheric properties over the specified altitude range and generates plots for:

- Temperature vs Altitude
- Pressure vs Altitude
- Density vs Altitude
- Combined Atmospheric Parameters
